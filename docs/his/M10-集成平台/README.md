# M10-集成平台 - 模块文档索引

> **模块编号**: M10
> **模块名称**: 集成平台子系统
> **优先级**: P1 - 重要功能
> **预估工作量**: 5人月

---

## 1. 模块概述

集成平台子系统提供院内/院间互联互通能力，实现EMPI患者主索引、主数据管理、消息引擎、接口适配器。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| EMPI管理 | 患者主索引创建、重复检测、合并、查询 | P0 |
| 主数据管理 | 科室、人员、诊断编码同步、分发 | P0 |
| 消息引擎 | HL7消息解析、FHIR资源转换、消息路由 | P0 |
| 接口适配 | 接口配置、日志、监控 | P0 |
