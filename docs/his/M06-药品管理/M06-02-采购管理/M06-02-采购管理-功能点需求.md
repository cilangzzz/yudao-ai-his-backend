## 1. 功能概述

### 1.1 功能定位

采购管理是药品管理子系统的供应链模块，负责采购计划生成、采购订单管理、供应商管理、采购验收等功能。系统支持智能采购计划生成，实现药品供应链的规范化管理。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 效率目标 | 采购响应效率 | 采购计划生成时间≤30分钟 |
| 准确目标 | 采购计划准确 | 采购计划准确率≥90% |
| 成本目标 | 采购成本控制 | 采购成本优化率≥5% |
| 合规目标 | 供应商资质合规 | 供应商资质合格率100% |

### 1.3 功能范围


## 4. 接口设计

### 4.1 采购计划生成接口

**接口路径**: `POST /api/pharm/purchase/plan/generate`

**请求参数**:
```json
{
  "planPeriod": "monthly",
  "planMonth": "2026-07",
  "calcMethod": "safety_stock",
  "safetyFactor": 1.5
}
```

**响应结果**:
```json
{
  "code": 200,
  "data": {
    "planId": 10001,
    "planItems": [
      {"drugId": 1001, "drugName": "阿莫西林胶囊", "suggestQty": 300}
    ]
  }
}
```
