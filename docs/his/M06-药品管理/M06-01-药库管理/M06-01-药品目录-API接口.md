# 药品目录 API 接口文档

> **模块**: 药库管理
> **Controller**: `HisDrugController`
> **基础路径**: `/his/drug`
> **版本**: v1.0
> **更新日期**: 2026-06-18

---

## 接口概览

| 接口 | 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|------|
| 创建药品目录 | POST | `/create` | `his:drug:create` | 新增药品 |
| 更新药品目录 | PUT | `/update` | `his:drug:update` | 更新药品信息 |
| 删除药品目录 | DELETE | `/delete` | `his:drug:delete` | 删除药品 |
| 获取药品详情 | GET | `/get` | `his:drug:query` | 根据ID查询药品 |
| 获取药品分页 | GET | `/page` | `his:drug:query` | 分页查询药品 |
| 获取药品列表 | GET | `/list` | `his:drug:query` | 批量查询药品 |
| 按名称模糊查询 | GET | `/list-by-name` | `his:drug:query` | 按名称搜索药品 |
| 按类型查询 | GET | `/list-by-type` | `his:drug:query` | 按药品类型查询 |
| 特殊药品列表 | GET | `/special-list` | `his:drug:query` | 获取麻精毒药品 |
| 抗菌药物列表 | GET | `/antibiotic-list` | `his:drug:query` | 获取抗菌药物 |
| 导出 Excel | GET | `/export-excel` | `his:drug:export` | 导出药品目录 |

---

## 接口详情

### 1. 创建药品目录

**请求**
- 方法: `POST`
- 路径: `/his/drug/create`
- 权限: `his:drug:create`
- Content-Type: `application/json`

**请求体** (`HisDrugSaveReqVO`)

```json
{
  "drugCode": "DRUG001",
  "drugName": "阿莫西林胶囊",
  "drugAlias": "阿莫西林",
  "drugType": 1,
  "drugCategory": 1,
  "spec": "0.5g*24粒",
  "unit": "盒",
  "minUnit": "粒",
  "packRatio": 24,
  "manufacturer": "XX制药",
  "approvalNo": "国药准字H12345678",
  "retailPrice": 25.00,
  "purchasePrice": 15.00,
  "insuranceType": 1,
  "insuranceCode": "YB001",
  "isSpecial": 0,
  "isAntibiotic": 1,
  "skinTestRequired": 0,
  "storageCondition": "常温",
  "shelfLife": 24,
  "status": 1,
  "remark": ""
}
```

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": 1  // 药品ID
}
```

### 3. 获取药品分页

**请求**
- 方法: `GET`
- 路径: `/his/drug/page`
- 权限: `his:drug:query`

**参数** (`HisDrugPageReqVO`)

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNo | Integer | 是 | 页码 |
| pageSize | Integer | 是 | 每页条数 |
| drugCode | String | 否 | 药品编码 |
| drugName | String | 否 | 药品名称（模糊） |
| drugType | Integer | 否 | 药品类型 |
| drugCategory | Integer | 否 | 药品分类 |
| manufacturer | String | 否 | 生产厂家 |
| status | Integer | 否 | 状态 |
| isSpecial | Integer | 否 | 是否特殊药品 |
| isAntibiotic | Integer | 否 | 是否抗菌药物 |

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "list": [...],
    "total": 100
  }
}
```

---

### 4. 按名称模糊查询

**请求**
- 方法: `GET`
- 路径: `/his/drug/list-by-name`
- 权限: `his:drug:query`

**参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| drugName | String | 是 | 药品名称（支持模糊） |

**响应**

返回匹配的药品列表

---

### 5. 获取特殊药品列表

**请求**
- 方法: `GET`
- 路径: `/his/drug/special-list`
- 权限: `his:drug:query`

**响应**

返回麻醉药品、精神药品、毒性药品列表

## 错误码

| 错误码 | 说明 |
|--------|------|
| 1_017_000_000 | 药品不存在 |
| 1_017_000_001 | 药品编码已存在 |
| 1_017_000_002 | 药品已停用 |
| 1_017_000_003 | 药品已过期 |

## 前端对接示例

### 查询药品

```javascript
const getDrugPage = async (params) => {
  const res = await request({
    url: '/his/drug/page',
    method: 'GET',
    params
  })
  return res.data
}
```

### 按名称搜索药品

```javascript
const searchDrugByName = async (drugName) => {
  const res = await request({
    url: '/his/drug/list-by-name',
    method: 'GET',
    params: { drugName }
  })
  return res.data
}
```

### 获取抗菌药物列表

```javascript
const getAntibioticList = async () => {
  const res = await request({
    url: '/his/drug/antibiotic-list',
    method: 'GET'
  })
  return res.data
}
```
