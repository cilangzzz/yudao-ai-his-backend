## 1. 功能概述

### 1.1 功能定位

处方审核与合理用药模块是药品管理子系统的核心安全模块，提供处方合理性自动审核、药物相互作用检查、过敏检查、剂量合理性检查、配伍禁忌检查等功能。系统集成了CDS（临床决策支持）引擎，在处方开立和审核阶段自动进行安全校验。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 安全目标 | 保障用药安全 | 不合理处方拦截率≥99% |
| 效率目标 | 提升审核效率 | 自动审核通过率≥85% |
| 覆盖目标 | CDS校验全覆盖 | 高风险药物检查覆盖率100% |
| 质量目标 | 减少用药差错 | 药物不良事件发生率≤0.1% |

## 4. 接口设计

### 4.1 处方审核接口

**接口路径**: `POST /api/pharm/audit`

**请求参数**:
```json
{
  "prescriptionId": 10001,
  "patientId": 1001,
  "diagnosisCode": "J18.9",
  "prescriptionItems": [
    {
      "drugId": 1001,
      "drugName": "阿莫西林胶囊",
      "dosage": "0.5g",
      "frequency": "tid",
      "days": 8
    }
  ]
}
```

**响应结果**:
```json
{
  "code": 200,
  "data": {
    "auditId": 20001,
    "passed": false,
    "warnings": [
      {
        "level": "HIGH",
        "type": "ALLERGY",
        "drugName": "阿莫西林胶囊",
        "message": "患者有青霉素过敏史",
        "suggestion": "更换为阿奇霉素或左氧氟沙星",
        "alternatives": [
          {"drugId": 2001, "drugName": "阿奇霉素片", "dosage": "0.5g qd"},
          {"drugId": 2002, "drugName": "左氧氟沙星片", "dosage": "0.5g qd"}
        ]
      }
    ],
    "auditTime": "2026-06-17 09:35:00"
  }
}
```

## 5. 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| BR-PHARM-005 | 处方审核流程 | 所有处方必须经过合理性审核方可调配 |
| BR-PHARM-006 | 相互作用检查 | 药物-药物相互作用检查必须覆盖所有已知高风险组合 |
| BR-PHARM-007 | 抗菌药物分级 | 抗菌药物使用需符合分级管理规定 |
| BR-PHARM-008 | 过敏警告强制处理 | 高风险过敏警告必须处理后才能继续 |
| BR-PHARM-009 | 肾功能剂量调整 | 肾功能不全患者(Ccr<30)必须进行剂量调整校验 |
