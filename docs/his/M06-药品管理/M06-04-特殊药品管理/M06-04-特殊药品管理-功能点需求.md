## 1. 功能概述

### 1.1 功能定位

特殊药品管理是药品管理子系统的专项安全模块，负责麻醉药品、精神药品、毒性药品的特殊管理。系统实现**"五专管理"（专人、专柜、专锁、专账、专方）**，确保特殊药品全流程合规可追溯。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 安全目标 | 特殊药品安全 | 特殊药品差错率0% |
| 合规目标 | 五专管理合规 | 五专管理合规率100% |
| 可追溯目标 | 全流程可追溯 | 特殊药品追溯覆盖率100% |
| 监管目标 | 监管报表合规 | 监管报表提交及时率100% |

### 1.3 功能范围


## 4. 接口设计

### 4.1 特殊药品出库登记接口

**接口路径**: `POST /api/pharm/special/outbound`

**请求参数**:
```json
{
  "drugId": 2001,
  "drugName": "盐酸吗啡注射液",
  "batchNo": "20260601",
  "quantity": 10,
  "unit": "mg",
  "prescriptionId": 10001,
  "patientId": 1001,
  "doctorId": 100,
  "dispensePharmacist": 1001,
  "issuePharmacist": 1002,
  "verifyPharmacist": 1003
}
```

**响应结果**:
```json
{
  "code": 200,
  "msg": "麻醉药品出库登记成功",
  "data": {
    "outboundId": 50001,
    "remainingStock": 50
  }
}
```

## 5. 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| BR-PHARM-009 | 麻醉药品五专 | 麻醉药品必须"五专"管理 |
| BR-PHARM-010 | 双人双锁 | 麻醉药品实行双人双锁管理 |
| BR-PHARM-011 | 专用处方 | 麻醉药品必须使用专用处方（红色处方） |
| BR-PHARM-012 | 三级复核 | 出库必须三人复核签字 |
| BR-PHARM-013 | 日清月结 | 麻醉药品必须每日清点、每月结算 |
