## 1. 模块概述

药品管理子系统覆盖药品全生命周期：入库→库存管理→出库→配药→发药，重点实现闭环给药和合理用药审核。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 药库管理 | 入库验收、出库、盘点、效期管理 | P0 |
| 药品目录 | 药品信息维护、分类、医保对照 | P0 |
| 药房库存 | 请领、库存查询、预警 | P0 |
| 处方审核 | 合理用药审核、相互作用检查 | P0 |
| CDS引擎 | 药物相互作用、过敏检查、剂量校验 | P0 |
