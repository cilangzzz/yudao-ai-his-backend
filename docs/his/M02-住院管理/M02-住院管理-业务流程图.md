## 1. 住院就诊全流程

**流程说明**：患者从入院登记到出院结算的完整住院过程。

```
流程概览：
入院登记 → 入科 → 医嘱开立 → 检查检验 → 治疗护理 → 手术(如需) → 出院申请 → 出院结算
```

### 1.1 住院主流程

```mermaid
flowchart TD
    A([医生开具入院证]) --> B[患者办理入院]
    B --> C[入院登记]
    C --> D[缴纳预交金]
    D --> E[医保登记]
    E --> F[分配床位]
    F --> G[入科确认]
    G --> H[入院护理评估]
    H --> I[医生开立医嘱]
    I --> J[护士审核医嘱]
    J --> K{医嘱类型}
    K -- 药品医嘱 --> L[药房配药]
    L --> M[护士给药执行]
    K -- 检验医嘱 --> N[标本采集]
    N --> O[检验科检验]
    O --> P[结果报告]
    K -- 检查医嘱 --> Q[预约检查]
    Q --> R[执行检查]
    R --> S[检查报告]
    K -- 护理医嘱 --> T[护理执行]
    K -- 手术医嘱 --> U[手术流程]
    M --> V[费用自动记账]
    P --> V
    S --> V
    T --> V
    U --> V
    V --> W{是否需要出院?}
    W -- 否 --> I
    W -- 是 --> X[出院申请]
    X --> Y[出院审批]
    Y --> Z[出院带药]
    Z --> AA[出院结算]
    AA --> AB[医保结算]
    AB --> AC[打印出院小结]
    AC --> AD[释放床位]
    AD --> AE([患者出院])
```

### 1.2 医嘱处理流程

```mermaid
flowchart TD
    A([医生开立医嘱]) --> B[选择医嘱类型]
    B --> C{医嘱类别}
    C -- 药品医嘱 --> D[选择药品]
    D --> E[设置用法用量]
    E --> F[CDS自动校验]
    F --> G{校验结果}
    G -- 有警告 --> H[显示警告信息]
    H --> I{医生决定}
    I -- 继续 --> J[确认开立]
    I -- 修改 --> D
    G -- 通过 --> J
    C -- 检验医嘱 --> K[选择检验项目]
    K --> J
    C -- 检查医嘱 --> L[选择检查项目]
    L --> J
    C -- 护理医嘱 --> M[选择护理项目]
    M --> J
    J --> N[医嘱提交]
    N --> O{长期/临时?}
    O -- 长期医嘱 --> P[长期医嘱生效]
    O -- 临时医嘱 --> Q[临时医嘱立即执行]
    P --> R[护士站接收]
    Q --> R
    R --> S[护士审核]
    S --> T{审核结果}
    T -- 通过 --> U[医嘱执行]
    T -- 退回 --> V[退回原因]
    V --> W[医生修改]
    W --> N
    U --> X[执行记录]
    X --> Y([医嘱完成])
```

### 1.3 手术流程

```mermaid
flowchart TD
    A([手术申请]) --> B[术前检查]
    B --> C[术前评估]
    C --> D{评估结果}
    D -- 可手术 --> E[手术排期]
    D -- 暂缓 --> F[术前准备]
    F --> B
    E --> G[手术室安排]
    G --> H[术前访视]
    H --> I[知情同意签署]
    I --> J[术前准备]
    J --> K[患者接入手术室]
    K --> L[手术安全核查]
    L --> M[麻醉诱导]
    M --> N[手术实施]
    N --> O[术中记录]
    O --> P[手术结束]
    P --> Q[麻醉恢复]
    Q --> R[术后交接]
    R --> S[术后医嘱]
    S --> T[术后护理]
    T --> U[术后访视]
    U --> V([手术流程结束])
```

### 1.4 出院结算流程

```mermaid
flowchart TD
    A([出院申请]) --> B[医生审核]
    B --> C[护士站确认]
    C --> D[停止长期医嘱]
    D --> E[出院带药开立]
    E --> F[费用汇总]
    F --> G[费用审核]
    G --> H{是否有未结费用?}
    H -- 是 --> I[补充计费]
    I --> F
    H -- 否 --> J{是否医保?}
    J -- 是 --> K[医保出院结算]
    K --> L[医保报销计算]
    L --> M[个人自付计算]
    J -- 否 --> M
    M --> N[显示结算清单]
    N --> O[患者确认]
    O --> P[收取个人费用]
    P --> Q[打印结算发票]
    Q --> R[打印出院小结]
    R --> S[打印出院带药]
    S --> T[释放床位]
    T --> U[病案归档]
    U --> V([出院完成])
```

## 2. 闭环药物管理流程（HIMSS EMRAM Stage 5要求）

```mermaid
flowchart TD
    A([医嘱开立]) --> B[药师审核]
    B --> C{审核通过?}
    C -- 否 --> D[退回修改]
    C -- 是 --> E[药房配药]
    E --> F[配药完成-条码贴签]
    F --> G[药品配送至病区]
    G --> H[护士接收核对]
    H --> I[执行给药]
    I --> J[扫描患者腕带]
    J --> K{身份匹配?}
    K -- 是 --> L[扫描药品条码]
    K -- 否 --> M[停止给药-身份核实]
    M --> J
    L --> N{药品匹配?}
    N -- 是 --> O[确认给药]
    O --> P[记录eMAR]
    P --> Q[费用自动记账]
    Q --> R([给药完成])
    N -- 否 --> S[停止给药-药品核实]
    S --> T[联系药师确认]
    T --> L
```

## 3. 护理工作流程

### 3.1 护士日常工作流程

```mermaid
flowchart TD
    A([交接班]) --> B[查看患者信息]
    B --> C[接收新医嘱]
    C --> D{医嘱类型}
    D -- 给药医嘱 --> E[配药-核对]
    E --> F[床旁给药-扫描腕带]
    F --> G[记录eMAR]
    D -- 检验医嘱 --> H[采集标本]
    H --> I[条码扫描确认]
    D -- 检查医嘱 --> J[陪同检查]
    D -- 护理医嘱 --> K[执行护理操作]
    G --> L[护理记录]
    I --> L
    J --> L
    K --> L
    L --> M[巡视病房]
    M --> N[记录生命体征]
    N --> O[异常情况处理]
    O --> P{是否需要报告医生?}
    P -- 是 --> Q[通知医生]
    Q --> R[执行医嘱调整]
    P -- 否 --> S[继续巡视]
    R --> S
    S --> T[书写护理记录]
    T --> U[交接班准备]
    U --> V[交接班记录]
    V --> W([交接班完成])
```

### 3.2 护理评估流程

```mermaid
flowchart TD
    A([入院]) --> B[入院护理评估]
    B --> C[跌倒风险评估]
    B --> D[压疮风险评估]
    B --> E[营养风险评估]
    B --> F[疼痛评估]
    C --> G{跌倒高风险?}
    G -- 是 --> H[跌倒预防措施]
    G -- 否 --> I[常规护理]
    D --> J{压疮高风险?}
    J -- 是 --> K[压疮预防措施]
    J -- 否 --> I
    E --> L{营养风险?}
    L -- 是 --> M[营养干预]
    L -- 否 --> I
    F --> N{疼痛评分}
    N -- 重度 --> O[疼痛处理]
    N -- 轻度 --> P[观察记录]
    N -- 无 --> I
    H --> Q[定期复评]
    K --> Q
    M --> Q
    O --> Q
    P --> Q
    I --> Q
    Q --> R([评估持续])
```
