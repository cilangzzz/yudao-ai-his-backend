## 1. 功能概述

### 1.1 功能定位

床位管理是住院管理子系统的资源管理模块，提供床位状态可视化总览、床位分配、床位调整、床位预约、床位维护等功能。系统支持全院床位统一管理，实现床位资源的优化配置。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 效率目标 | 床位分配效率 | 床位分配时间≤5分钟 |
| 利用目标 | 床位利用率 | 床位利用率≥85% |
| 可视目标 | 床位状态可视 | 床位状态实时更新 |
| 管理目标 | 床位周转率 | 平均住院日≤10天 |

## 4. 接口设计

### 4.1 床位查询接口

**接口路径**: `GET /api/ip/bed/overview`

**响应结果**:
```json
{
  "code": 200,
  "data": {
    "total": 500,
    "available": 75,
    "occupied": 410,
    "maintenance": 15,
    "utilizationRate": 82.0,
    "wards": [
      {
        "wardId": 101,
        "wardName": "内科一病区",
        "total": 40,
        "available": 5,
        "occupied": 35
      }
    ]
  }
}
```

### 4.2 床位分配接口

**接口路径**: `POST /api/ip/bed/assign`

**请求参数**:
```json
{
  "admissionId": 10001,
  "bedId": 102
}
```

**响应结果**:
```json
{
  "code": 200,
  "msg": "床位分配成功",
  "data": {
    "bedNo": "101B",
    "assignTime": "2026-06-17 10:00:00"
  }
}
```
