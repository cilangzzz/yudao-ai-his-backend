# 医嘱管理 API 接口文档

> **模块**: 住院医生工作站
> **Controller**: `HisOrderController`
> **基础路径**: `/his/order`
> **版本**: v1.0
> **更新日期**: 2026-06-18

---

## 接口概览

| 接口 | 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|------|
| 获取医嘱详情 | GET | `/get` | `his:order:query` | 根据ID查询医嘱 |
| 获取医嘱分页 | GET | `/page` | `his:order:query` | 分页查询医嘱列表 |
| 获取住院医嘱列表 | GET | `/list-by-admission` | `his:order:query` | 查询住院的所有医嘱 |
| 获取待审核医嘱 | GET | `/pending-audit-list` | `his:order:query` | 查询待审核的医嘱 |
| 获取执行中医嘱 | GET | `/executing-list` | `his:order:query` | 查询正在执行的医嘱 |
| 获取长期医嘱 | GET | `/long-term-list` | `his:order:query` | 查询长期医嘱列表 |
| 获取药品医嘱 | GET | `/drug-order-list` | `his:order:query` | 查询药品类医嘱 |
| 审核医嘱 | POST | `/audit` | `his:order:audit` | 审核医嘱 |
| 执行医嘱 | POST | `/execute` | `his:order:execute` | 执行医嘱 |
| 停止医嘱 | POST | `/stop` | `his:order:stop` | 停止医嘱 |
| 作废医嘱 | POST | `/cancel` | `his:order:cancel` | 作废医嘱 |

### 2. 审核医嘱

**请求**
- 方法: `POST`
- 路径: `/his/order/audit`
- 权限: `his:order:audit`
- Content-Type: `application/json`

**请求体** (`HisOrderAuditReqVO`)

```json
{
  "orderId": 1,
  "auditDoctorId": 101,
  "auditDoctorName": "李医生",
  "auditResult": 1,
  "auditRemark": ""
}
```

**参数说明**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| orderId | Long | 是 | 医嘱ID |
| auditDoctorId | Long | 是 | 审核医生ID |
| auditDoctorName | String | 是 | 审核医生姓名 |
| auditResult | Integer | 是 | 审核结果：1-通过，2-不通过 |
| auditRemark | String | 否 | 审核备注 |

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": true
}
```

## 医嘱状态流转

```
待审核(1) → 已审核(2) → 执行中(3) → 已停止(4)
    ↓           ↓           ↓
  已作废(5)   已作废(5)   已完成(6)
```

| 状态值 | 说明 | 可操作 |
|--------|------|--------|
| 1 | 待审核 | 审核、作废 |
| 2 | 已审核 | 执行、作废 |
| 3 | 执行中 | 停止 |
| 4 | 已停止 | 无 |
| 5 | 已作废 | 无 |
| 6 | 已完成 | 无 |

---

## 医嘱类型

| 值 | 说明 |
|----|------|
| 1 | 长期医嘱 |
| 2 | 临时医嘱 |

## 错误码

| 错误码 | 说明 |
|--------|------|
| 1_016_003_000 | 医嘱不存在 |
| 1_016_003_001 | 医嘱已停止 |
| 1_016_003_002 | 医嘱已完成 |
| 1_016_003_003 | 医嘱已作废 |
| 1_016_003_004 | 医嘱明细不存在 |
| 1_016_003_005 | 医嘱已执行，无法修改 |

## 前端对接示例

### 获取住院医嘱列表

```javascript
const getOrderList = async (admissionId) => {
  const res = await request({
    url: '/his/order/list-by-admission',
    method: 'GET',
    params: { admissionId }
  })
  return res.data
}
```

### 审核医嘱

```javascript
const auditOrder = async (data) => {
  const res = await request({
    url: '/his/order/audit',
    method: 'POST',
    data
  })
  return res.data
}
```

### 停止医嘱

```javascript
const stopOrder = async (orderId, stopDoctorId, stopDoctorName, stopReason) => {
  const res = await request({
    url: '/his/order/stop',
    method: 'POST',
    data: { orderId, stopDoctorId, stopDoctorName, stopReason }
  })
  return res.data
}
```
