## 1. 功能概述

### 1.1 功能定位

住院医生工作站是住院医生的核心工作平台，提供医嘱开立（CPOE）、病历书写、诊断管理、会诊申请、手术申请等功能。系统集成了CDS（临床决策支持）功能，在医嘱开立时自动进行安全校验。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 效率目标 | 提升医嘱开立效率 | 医嘱开立时间≤5分钟 |
| 安全目标 | CDS校验全覆盖 | CDS校验覆盖率100% |
| 规范目标 | 病历书写规范 | 入院记录24h完成率100% |
| 质量目标 | 医嘱执行及时 | 医嘱执行及时率≥95% |

## 4. 接口设计

### 4.1 医嘱开立接口

**接口路径**: `POST /api/ip/order`

**请求参数**:
```json
{
  "admissionId": 10001,
  "orderType": 1,
  "orders": [
    {
      "orderCategory": "drug",
      "drugId": 1001,
      "drugName": "0.9%氯化钠注射液",
      "dosage": "250",
      "dosageUnit": "ml",
      "frequency": "qd",
      "days": 7,
      "route": "ivgtt"
    }
  ]
}
```

**响应结果**:
```json
{
  "code": 200,
  "msg": "医嘱开立成功",
  "data": {
    "orderIds": [20001, 20002],
    "submitTime": "2026-06-17 10:00:00"
  }
}
```

## 5. 业务规则
