# 入院管理 API 接口文档

> **模块**: 入院管理
> **Controller**: `HisAdmissionController`
> **基础路径**: `/his/admission`
> **版本**: v1.0
> **更新日期**: 2026-06-18

---

## 接口概览

| 接口 | 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|------|
| 入院登记 | POST | `/create` | `his:admission:create` | 创建入院记录 |
| 更新入院记录 | PUT | `/update` | `his:admission:update` | 更新入院信息 |
| 删除入院记录 | DELETE | `/delete` | `his:admission:delete` | 删除入院记录 |
| 获取入院详情 | GET | `/get` | `his:admission:query` | 根据ID查询 |
| 根据住院号查询 | GET | `/get-by-no` | `his:admission:query` | 根据住院号查询 |
| 获取入院分页 | GET | `/page` | `his:admission:query` | 分页查询入院列表 |
| 患者历史入院记录 | GET | `/list-by-patient` | `his:admission:query` | 查询患者历史住院 |
| 在院患者列表 | GET | `/in-hospital-list` | `his:admission:query` | 获取所有在院患者 |
| 科室在院患者 | GET | `/in-hospital-by-dept` | `his:admission:query` | 按科室查询在院患者 |
| 病区在院患者 | GET | `/in-hospital-by-ward` | `his:admission:query` | 按病区查询在院患者 |
| 申请出院 | POST | `/apply-discharge` | `his:admission:discharge` | 提交出院申请 |
| 确认出院 | POST | `/confirm-discharge` | `his:admission:discharge` | 确认患者出院 |
| 转科 | POST | `/transfer-dept` | `his:admission:transfer` | 患者转科处理 |
| 更换床位 | POST | `/update-bed` | `his:admission:update` | 更换患者床位 |
| 紧急入院 | POST | `/emergency-admission` | `his:admission:create` | 紧急入院处理 |
| 增加预交金 | POST | `/add-deposit` | `his:admission:update` | 增加预交金 |
| 住院统计 | GET | `/statistics` | `his:admission:query` | 获取住院统计数据 |

---

## 接口详情

### 1. 入院登记

**请求**
- 方法: `POST`
- 路径: `/his/admission/create`
- 权限: `his:admission:create`
- Content-Type: `application/json`

**请求体** (`HisAdmissionSaveReqVO`)

```json
{
  "patientId": 1,
  "patientName": "张三",
  "idCardNo": "320102199001011234",
  "phone": "13800138000",
  "deptId": 10,
  "deptName": "内科",
  "wardId": 1,
  "wardName": "一病区",
  "bedId": 1,
  "bedNo": "01",
  "doctorId": 100,
  "doctorName": "王医生",
  "nurseId": 200,
  "nurseName": "张护士",
  "admissionWay": 1,
  "admissionCondition": 1,
  "diagnosis": "肺炎",
  "diagnosisCode": "J18.9",
  "allergyHistory": "青霉素过敏",
  "depositAmount": 5000.00,
  "remark": ""
}
```

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": 1  // 入院记录ID
}
```

### 4. 转科

**请求**
- 方法: `POST`
- 路径: `/his/admission/transfer-dept`
- 权限: `his:admission:transfer`

**参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | Long | 是 | 入院记录ID |
| newDeptId | Long | 是 | 新科室ID |
| newWardId | Long | 是 | 新病区ID |
| newBedId | Long | 否 | 新床位ID |

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": true
}
```

## 入院状态

| 状态值 | 说明 |
|----|------|
| 0 | 待入院 |
| 1 | 在院 |
| 2 | 待出院 |
| 3 | 已出院 |

---

## 入院方式

| 值 | 说明 |
|----|------|
| 1 | 门诊入院 |
| 2 | 急诊入院 |
| 3 | 转院入院 |
| 4 | 其他 |

## 入院病情

| 值 | 说明 |
|----|------|
| 1 | 一般 |
| 2 | 急 |
| 3 | 危重 |

## 错误码

| 错误码 | 说明 |
|--------|------|
| 1_016_000_000 | 入院记录不存在 |
| 1_016_000_001 | 患者已出院 |
| 1_016_000_002 | 床位不可用 |
| 1_016_000_003 | 预交金不足 |

## 前端对接示例

### 入院登记

```javascript
const createAdmission = async (data) => {
  const res = await request({
    url: '/his/admission/create',
    method: 'POST',
    data
  })
  return res.data
}
```

### 获取病区在院患者

```javascript
const getInHospitalByWard = async (wardId) => {
  const res = await request({
    url: '/his/admission/in-hospital-by-ward',
    method: 'GET',
    params: { wardId }
  })
  return res.data
}
```

### 转科处理

```javascript
const transferDept = async (id, newDeptId, newWardId, newBedId) => {
  const res = await request({
    url: '/his/admission/transfer-dept',
    method: 'POST',
    params: { id, newDeptId, newWardId, newBedId }
  })
  return res.data
}
```
