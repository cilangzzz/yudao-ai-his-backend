## 1. 模块概述

运营管理子系统面向医院管理者，提供运营数据分析、绩效管理、决策支持。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 运营看板 | 门诊量、住院量、床位使用率、收入统计 | P0 |
| 统计报表 | 科室工作量、医生工作量、药品消耗、手术统计 | P0 |
| 绩效管理 | 绩效指标配置、评分、报表 | P1 |
| 经营分析 | 收入分析、成本分析 | P2 |
