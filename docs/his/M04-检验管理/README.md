## 1. 模块概述

检验管理子系统(LIS)覆盖检验全流程：申请→标本采集→检验→报告发布，重点实现危急值15分钟通报机制。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 检验申请 | 申请接收、查询、状态追踪 | P0 |
| 标本管理 | 采集、条码生成、运送、接收、拒收 | P0 |
| 检验执行 | 任务分配、仪器对接、结果录入、审核 | P0 |
| 报告管理 | 报告生成、审核发布、危急值通报 | P0 |
| 检验项目 | 项目配置、参考范围、危急值阈值 | P0 |
