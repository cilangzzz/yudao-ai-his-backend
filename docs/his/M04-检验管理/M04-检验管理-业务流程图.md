## 1. 检验全流程

```mermaid
flowchart TD
    A([医生开具检验申请]) --> B[申请单发送至LIS]
    B --> C[生成标本条码]
    C --> D[护士采集标本]
    D --> D1[扫描条码-采集确认]
    D1 --> E[标本运送]
    E --> F[检验科接收标本]
    F --> G{标本合格?}
    G -- 否 --> H[拒收-通知重新采集]
    H --> D
    G -- 是 --> I[标本编号]
    I --> J[上机检测]
    J --> K[结果自动采集]
    K --> L{结果异常?}
    L -- 危急值 --> M[危急值报告]
    M --> N[通知临床科室]
    N --> O[临床确认接收]
    L -- 正常 --> P[结果审核]
    O --> P
    P --> Q{审核通过?}
    Q -- 是 --> R[结果发布]
    R --> S[报告生成]
    S --> T[报告发送至HIS]
    T --> U[医生查看结果]
    U --> V([检验完成])
    Q -- 否 --> W[复检/复查]
    W --> J
```
