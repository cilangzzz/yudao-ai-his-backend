## 1. 模块概述

影像管理子系统(RIS/PACS)覆盖影像检查全流程：申请→检查→影像存储→报告发布，遵循DICOM标准，支持三级存储架构。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 影像申请 | 申请接收、检查排期 | P0 |
| 影像检查 | 检查登记、DICOM影像采集、上传 | P0 |
| 影像存储 | 在线存储、归档、迁移 | P0 |
| 影像查看 | 影像调阅、窗宽窗位、测量标注 | P0 |
| 影像报告 | 报告书写、审核发布 | P0 |
