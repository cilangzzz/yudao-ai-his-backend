> **预估工作量**: 7人月

---

## 1. 模块概述

手术麻醉子系统覆盖手术全流程：申请→排期→术前准备→手术执行→麻醉记录→术后管理。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 手术申请 | 申请接收、审核、手术分级校验 | P0 |
| 手术排期 | 手术室管理、排期、调整、术前清单 | P0 |
| 手术执行 | 手术登记、术前核查、过程记录、用药 | P0 |
| 麻醉管理 | 麻醉方案、用药记录、生命体征、ASA评分 | P0 |
| 术后管理 | 术后观察、随访 | P0 |
