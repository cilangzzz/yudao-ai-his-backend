## 1. 模块概述

电子病历子系统管理全院病历文书的创建、审签、归档、质控，支持门诊病历和住院病历，病历保存期限≥30年。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 病历模板 | 模板创建、分类、权限、变量 | P0 |
| 病历编辑 | 入院记录、病程记录、出院记录、知情同意书 | P0 |
| 病历审签 | 提交审签、上级医师审核、电子签名 | P0 |
| 病历质控 | 时限质控(24h/8h)、完整性质控 | P0 |
| 病历归档 | 归档提交、锁定、病案检索、封存 | P0 |
