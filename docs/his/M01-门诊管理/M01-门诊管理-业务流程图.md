## 1. 门诊就诊全流程

**流程说明**：患者从挂号到取药完成一次完整的门诊就诊过程。

```
流程概览：
患者到达 → 挂号/签到 → 候诊 → 就诊 → 缴费 → 取药/检查 → 离院
```

### 1.1 门诊就诊主流程

```mermaid
flowchart TD
    A([患者到达医院]) --> B{是否已预约?}
    B -- 是 --> C[预约签到]
    B -- 否 --> D[现场挂号]
    D --> E[选择科室/医生]
    E --> F[缴纳挂号费]
    F --> G[分配排队号]
    C --> G
    G --> H[进入候诊队列]
    H --> I[叫号就诊]
    I --> J[医生接诊]
    J --> K[书写门诊病历]
    K --> L[开具诊断]
    L --> M{是否需要检查检验?}
    M -- 是 --> N[开具检查/检验申请]
    N --> O[患者缴费]
    O --> P[执行检查/检验]
    P --> Q[获取结果]
    Q --> R[医生查看结果]
    M -- 否 --> R
    R --> S{是否需要开药?}
    S -- 是 --> T[开具处方]
    T --> U[处方审核-CDS]
    U --> V[患者缴费]
    V --> W[药房取药]
    S -- 否 --> X
    W --> X[就诊结束]
    X --> Y([患者离院])
```

### 1.2 挂号流程详细

```mermaid
flowchart TD
    A([开始]) --> B{挂号方式}
    B -- 线上预约 --> C[选择就诊日期]
    C --> D[选择科室]
    D --> E[选择医生/时段]
    E --> F[确认预约信息]
    F --> G[在线支付挂号费]
    G --> H[生成预约挂号单]
    H --> I([预约成功])

    B -- 现场挂号 --> J[出示身份证/医保卡]
    J --> K[读取患者信息]
    K --> L{是否已建档?}
    L -- 否 --> M[新建患者档案]
    M --> N[录入基本信息]
    N --> O[分配患者ID]
    L -- 是 --> O
    O --> P[选择科室]
    P --> Q[选择挂号类型]
    Q --> R[选择医生]
    R --> S[计算挂号费]
    S --> T{是否医保?}
    T -- 是 --> U[医保身份验证]
    U --> V[医保挂号费计算]
    T -- 否 --> V
    V --> W[收取挂号费]
    W --> X[打印挂号单]
    X --> Y[分配排队号]
    Y --> Z([挂号完成])

    B -- 自助机挂号 --> AA[刷卡/扫码]
    AA --> AB[选择科室医生]
    AB --> AC[自助缴费]
    AC --> AD[打印挂号单]
    AD --> Z
```

### 1.3 门诊收费流程

```mermaid
flowchart TD
    A([收费开始]) --> B[获取待收费项目]
    B --> C[汇总费用明细]
    C --> D[费用分类统计]
    D --> E{是否医保患者?}
    E -- 是 --> F[医保费用预结算]
    F --> G[计算医保报销金额]
    G --> H[计算个人自付金额]
    E -- 否 --> H
    H --> I[显示收费清单]
    I --> J[患者确认]
    J --> K{支付方式}
    K -- 现金 --> L[现金收款]
    K -- 医保 --> M[医保结算]
    K -- 微信/支付宝 --> N[扫码支付]
    K -- 银行卡 --> O[POS刷卡]
    L --> P[打印发票]
    M --> P
    N --> P
    O --> P
    P --> Q[更新收费状态]
    Q --> R([收费完成])
```

## 2. 门诊处方调剂流程

```mermaid
flowchart TD
    A([接收处方]) --> B[处方审核]
    B --> C{审核结果}
    C -- 不合理 --> D[退回医生修改]
    D --> E[医生修改处方]
    E --> B
    C -- 合理 --> F[打印处方标签]
    F --> G[药品调配]
    G --> H[核对药品]
    H --> I[发药确认]
    I --> J[打印用药指导]
    J --> K[库存扣减]
    K --> L([发药完成])
```

## 3. 流程统计与监控指标

| 流程 | 关键指标 | 目标值 |
|------|----------|--------|
| 门诊就诊 | 平均就诊时间 | <= 30分钟 |
| 门诊缴费 | 缴费等待时间 | <= 5分钟 |
| 药品调剂 | 处方调剂时间 | <= 15分钟 |
