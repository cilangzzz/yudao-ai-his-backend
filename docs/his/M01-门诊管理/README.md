## 1. 模块概述

门诊管理子系统覆盖门诊全流程：挂号→候诊→接诊→开方→收费→取药，支持日均5000+门诊量。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 挂号管理 | 现场挂号、预约挂号、急诊挂号、退号 | P0 |
| 排班管理 | 号源管理、排班配置 | P0 |
| 医生工作站 | 接诊、诊断录入、处方开立、CDS校验 | P0 |
| 收费管理 | 费用汇总、医保结算、退费 | P0 |
| 药房管理 | 处方接收、审核、调配、发药 | P0 |
