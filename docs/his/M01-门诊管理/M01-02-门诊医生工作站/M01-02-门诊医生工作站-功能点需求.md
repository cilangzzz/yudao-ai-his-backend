## 1. 功能概述

### 1.1 功能定位

门诊医生工作站是门诊医生的核心工作平台，提供患者接诊、诊断开立、处方开立、检查检验申请、病历书写等功能。系统集成了CDS（临床决策支持）功能，在处方开立时自动进行药物相互作用、过敏检查、剂量合理性校验。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 效率目标 | 提升医生接诊效率 | 平均接诊时间≤15分钟 |
| 安全目标 | 保障用药安全 | CDS校验覆盖率100% |
| 规范目标 | 病历书写规范化 | ICD-10诊断编码使用率100% |
| 质量目标 | 提升诊断准确率 | 处方审核通过率≥95% |

## 4. 接口设计

### 4.1 接诊接口

**接口路径**: `POST /api/op/visit/{registerId}/accept`

**响应结果**:
```json
{
  "code": 200,
  "msg": "接诊成功",
  "data": {
    "visitId": 10001,
    "patientInfo": {
      "patientId": 1001,
      "name": "张三",
      "gender": "男",
      "age": 35,
      "allergyHistory": ["青霉素"],
      "medicalHistory": ["高血压病史5年"]
    },
    "recentVisits": [...]
  }
}
```

### 4.2 诊断开立接口

**接口路径**: `POST /api/op/visit/{visitId}/diagnosis`

**请求参数**:
```json
{
  "diagnosisList": [
    {
      "diagnosisCode": "J18.9",
      "diagnosisName": "肺炎（未特指）",
      "diagnosisOrder": 1,
      "diagnosisType": "门诊诊断"
    },
    {
      "diagnosisCode": "I10",
      "diagnosisName": "高血压",
      "diagnosisOrder": 2,
      "diagnosisType": "门诊诊断"
    }
  ]
}
```

### 4.3 处方开立接口

**接口路径**: `POST /api/op/prescription`

**请求参数**:
```json
{
  "visitId": 10001,
  "patientId": 1001,
  "prescriptionType": 1,
  "diagnosisCode": "J18.9",
  "prescriptionItems": [
    {
      "drugId": 1001,
      "drugName": "阿莫西林胶囊",
      "spec": "0.5g×24",
      "quantity": 24,
      "unit": "粒",
      "usage": "tid po",
      "frequency": "3次/日",
      "days": 8
    }
  ]
}
```

### 4.4 CDS校验接口

**接口路径**: `POST /api/cds/check`

**请求参数**:
```json
{
  "patientId": 1001,
  "prescriptionItems": [
    {"drugId": 1001, "drugName": "阿莫西林胶囊"},
    {"drugId": 1002, "drugName": "布洛芬片"}
  ]
}
```

**响应结果**:
```json
{

      {
        "level": "HIGH",
        "type": "ALLERGY",

        "level": "MEDIUM",

        "drugName": "布洛芬片",
        "message": "患者有高血压病史",
        "suggestion": "监测血压变化"
      }
    ],
    "passed": false
  }
}
```

## 5. 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| BR-OP-009 | 处方开立CDS校验 | 处方开立时必须进行CDS校验（药物相互作用、过敏检查、剂量合理性） |
| BR-OP-010 | 诊断ICD-10编码 | 诊断必须使用ICD-10编码 |
| BR-OP-011 | 处方金额确认 | 处方金额超过500元需二次确认 |
| BR-OP-012 | 病历完成时限 | 门诊病历必须在就诊完成后24小时内完成 |
| BR-OP-013 | 处方审核流程 | 处方提交后自动进入药房审核流程 |
