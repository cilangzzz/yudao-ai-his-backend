# 门诊费用 API 接口文档

> **模块**: 门诊收费管理
> **Controller**: `OpFeeController`
> **基础路径**: `/his/op-fee`
> **版本**: v1.0
> **更新日期**: 2026-06-18

---

## 接口概览

| 接口 | 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|------|
| 创建门诊费用 | POST | `/create` | `his:op-fee:create` | 创建费用单 |
| 更新门诊费用 | PUT | `/update` | `his:op-fee:update` | 更新费用信息 |
| 删除门诊费用 | DELETE | `/delete` | `his:op-fee:delete` | 删除费用单 |
| 获取门诊费用 | GET | `/get` | `his:op-fee:query` | 根据ID查询费用 |
| 获取费用分页 | GET | `/page` | `his:op-fee:query` | 分页查询费用 |
| 获取费用列表 | GET | `/list` | `his:op-fee:query` | 批量查询费用 |
| 根据挂号ID查询 | GET | `/get-by-register` | `his:op-fee:query` | 查询挂号费用 |
| 患者未收费费用 | GET | `/unpaid-by-patient` | `his:op-fee:query` | 查询患者待缴费 |
| 创建费用明细 | POST | `/item/create` | `his:op-fee:create` | 添加费用明细 |
| 更新费用明细 | PUT | `/item/update` | `his:op-fee:update` | 更新费用明细 |
| 删除费用明细 | DELETE | `/item/delete` | `his:op-fee:delete` | 删除费用明细 |
| 获取费用明细 | GET | `/item/get` | `his:op-fee:query` | 查询费用明细 |
| 费用明细分页 | GET | `/item/page` | `his:op-fee:query` | 分页查询明细 |
| 按费用ID查明细 | GET | `/item/list-by-fee` | `his:op-fee:query` | 查询费用明细列表 |
| 按来源查明细 | GET | `/item/list-by-source` | `his:op-fee:query` | 按来源查询明细 |

## 接口详情

### 1. 创建门诊费用

**请求**
- 方法: `POST`
- 路径: `/his/op-fee/create`
- 权限: `his:op-fee:create`
- Content-Type: `application/json`

**请求体** (`OpFeeSaveReqVO`)

```json
{
  "registerId": 1,
  "patientId": 1,
  "patientName": "张三",
  "deptId": 10,
  "deptName": "内科",
  "doctorId": 100,
  "doctorName": "王医生",
  "totalAmount": 150.00,
  "discountAmount": 0,
  "payAmount": 150.00,
  "remark": ""
}
```

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": 1  // 费用ID
}
```

### 3. 根据挂号ID获取费用

**请求**
- 方法: `GET`
- 路径: `/his/op-fee/get-by-register`
- 权限: `his:op-fee:query`

**参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| registerId | Long | 是 | 挂号ID |

**响应**

返回该挂号关联的费用单及明细

---

### 4. 创建费用明细

**请求**
- 方法: `POST`
- 路径: `/his/op-fee/item/create`
- 权限: `his:op-fee:create`
- Content-Type: `application/json`

**请求体** (`OpFeeItemSaveReqVO`)

```json
{
  "feeId": 1,
  "itemCode": "ITEM001",
  "itemName": "血常规",
  "itemType": 3,
  "itemCategory": "检验",
  "spec": "",
  "unit": "次",
  "quantity": 1,
  "unitPrice": 50.00,
  "totalAmount": 50.00,
  "discountAmount": 0,
  "payAmount": 50.00,
  "sourceType": 1,
  "sourceId": 1,
  "sourceNo": "LAB202406180001"
}
```

**响应**

```json
{
  "code": 0,
  "msg": "success",
  "data": 1  // 明细ID
}
```

---

### 5. 获取患者未收费费用

**请求**
- 方法: `GET`
- 路径: `/his/op-fee/unpaid-by-patient`
- 权限: `his:op-fee:query`

**参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| patientId | Long | 是 | 患者ID |

**响应**

返回该患者所有未收费的费用单列表

## 前端对接示例

### 创建门诊费用

```javascript
const createFee = async (data) => {
  const res = await request({
    url: '/his/op-fee/create',
    method: 'POST',
    data
  })
  return res.data
}
```

### 获取挂号费用

```javascript
const getFeeByRegister = async (registerId) => {
  const res = await request({
    url: '/his/op-fee/get-by-register',
    method: 'GET',
    params: { registerId }
  })
  return res.data
}
```

### 添加费用明细

```javascript
const addFeeItem = async (data) => {
  const res = await request({
    url: '/his/op-fee/item/create',
    method: 'POST',
    data
  })
  return res.data
}
```
