---

## 1. 模块概述

AI辅助子系统提供智慧医疗能力，包括智能分诊、影像AI辅助诊断、病历质控AI。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 智能分诊 | 症状录入、科室推荐、分诊建议 | P0 |
| 影像AI | 病灶识别、病灶测量、AI辅助报告 | P0 |
| 病历质控AI | 完整性检查、规范性检查、质控建议 | P0 |
| AI模型管理 | 模型配置、版本管理 | P0 |
