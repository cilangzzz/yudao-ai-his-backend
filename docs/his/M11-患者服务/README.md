## 1. 模块概述

患者服务子系统面向患者用户，提供患者门户、预约挂号、报告查询、在线缴费等自助服务。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 患者门户 | 患者注册登录、信息管理、就诊记录查询 | P0 |
| 预约挂号 | 号源查询、预约挂号、取消、提醒 | P0 |
| 报告查询 | 检验报告、影像报告、推送通知 | P0 |
| 在线缴费 | 待缴费查询、在线支付、支付记录 | P0 |
| 健康档案 | 健康档案查看、过敏史查看 | P1 |
