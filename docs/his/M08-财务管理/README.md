## 1. 模块概述

财务管理子系统覆盖全院费用管理：收费项目管理、医保结算、费用记账、财务报表。

### 1.1 核心功能

| 子模块 | 功能描述 | 优先级 |
|--------|----------|--------|
| 收费项目 | 项目维护、价格管理、医保对照 | P0 |
| 医保结算 | 医保身份验证、费用结算、对账 | P0 |
| 费用记账 | 门诊/住院费用明细记账 | P0 |
| 财务报表 | 日结、月结、收入统计 | P0 |
