## 1. 功能概述

### 1.1 功能定位

数据字典是系统管理子系统的基础数据模块，提供字典类型管理、字典数据管理、字典版本管理等功能。系统统一管理全院基础数据字典，包括诊断编码（ICD-10）、科室字典、收费项目等。

### 1.2 功能范围

```
M09-05 数据字典
├── 字典类型管理
│   ├── 字典类型创建
│   ├── 字典类型修改
│   ├── 字典类型删除
│   └── 字典类型查询
├── 字典数据管理
│   ├── 字典数据创建
│   ├── 字典数据修改
│   ├── 字典数据删除
│   └── 字典数据排序
├── 标准字典
│   ├── ICD-10诊断编码
│   ├── 科室字典
│   ├── 医保目录对照
│   └── 收费项目字典
└── 字典版本管理
    ├── 字典版本记录
    ├── 字典变更历史
    └── 字典回滚
```

## 4. 接口设计

### 4.1 字典查询接口

**接口路径**: `GET /api/sys/dict/{dictType}`

**响应结果**:
```json
{
  "code": 200,
  "data": [
    {"label": "男", "value": "1"},
    {"label": "女", "value": "2"},
    {"label": "未知", "value": "9"}
  ]
}
```
