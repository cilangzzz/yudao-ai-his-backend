## 1. 功能概述

### 1.1 功能定位

组织架构是系统管理子系统的基础数据模块，提供医院信息配置、科室管理、病区管理、岗位管理、人员信息维护等功能。系统实现医院组织架构的完整管理，为业务系统提供组织数据支撑。

### 1.2 功能范围

```
M09-04 组织架构
├── 医院信息
│   ├── 医院基本信息
│   ├── 医院资质信息
│   └── 医院联系方式
├── 科室管理
│   ├── 科室创建
│   ├── 科室修改
│   ├── 科室层级管理
│   └── 科室属性配置
├── 病区管理
│   ├── 病区创建
│   ├── 病区床位配置
│   └── 病区科室关联
├── 岗位管理
│   ├── 岗位定义
│   ├── 岗位职责
│   └── 岗位人员配置
└── 人员信息
    ├── 人员档案
    ├── 人员资质
    └── 人员排班
```

## 4. 接口设计

### 4.1 科室管理接口

**接口路径**: `POST /api/sys/dept`

**请求参数**:
```json
{
  "deptCode": "DEPT001",
  "deptName": "内科",
  "deptType": 1,
  "parentId": 0,
  "leaderId": 100,
  "phone": "0571-12345678"
}
```
