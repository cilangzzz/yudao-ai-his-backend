## 1. 功能概述

### 1.1 功能定位

权限管理是系统管理子系统的安全核心模块，提供菜单权限、按钮权限、数据权限的三级权限控制。系统基于RBAC模型，实现细粒度的权限管理，支持权限的灵活配置和动态调整。

### 1.2 功能范围

```
M09-03 权限管理
├── 菜单权限
│   ├── 菜单定义
│   ├── 菜单权限分配
│   └── 菜单权限查询
├── 按钮权限
│   ├── 按钮定义
│   ├── 按钮权限分配
│   └── 按钮权限校验
├── 数据权限
│   ├── 数据权限规则
│   ├── 数据权限分配
│   └── 数据权限过滤
└── 权限审计
    ├── 权限变更记录
    └── 权限使用日志
```

## 4. 接口设计

### 4.1 权限校验接口

**接口路径**: `POST /api/sys/permission/check`

**请求参数**:
```json
{
  "userId": 1001,
  "permissionCode": "op:register:add"
}
```

**响应结果**:
```json
{
  "code": 200,
  "data": {
    "hasPermission": true
  }
}
```
